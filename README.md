⚖️ LegalEase – Online Lawyer Hiring Platform

A modern MERN Stack web application that connects clients with professional lawyers. Users can browse legal experts, send hire requests, make secure payments, and manage their hiring history. Lawyers can showcase and manage their legal services, while admins oversee the entire platform.

🌐 Live Demo

🔗 Live Site: https://legal-ease-client-sooty.vercel.app/

---

📖 Project Description

LegalEase is a full-stack online lawyer hiring platform built with the MERN Stack. It provides a secure and user-friendly environment where clients can discover experienced lawyers, submit hiring requests, complete payments through Stripe, and leave reviews after hiring.

Lawyers can manage their legal profiles, accept or reject hiring requests, and update their services after completing a one-time verification process. Administrators have full control over users, lawyers, transactions, and platform analytics through a dedicated dashboard.

✨ Key Features

- 🔐 JWT Authentication with Email/Password & Google Login
- 👤 Role-Based Dashboard (User, Lawyer & Admin)
- ⚖️ Browse, Search, Filter & Paginate Lawyers
- 📄 Detailed Lawyer Profile Page
- 💼 Hire Request Management System
- 💳 Secure Stripe Payment Integration
- 💬 Comment System (Only Hired Users Can Comment)
- 📝 Lawyer Service CRUD Operations
- 📊 Admin Analytics Dashboard
- 👥 User & Role Management
- 📜 Transaction History
- 🖼️ Image Upload using imgBB
- 📱 Fully Responsive Design
- ⚡ Loading Skeletons & Error Handling
- 🎨 Smooth Animations with Framer Motion
